Aula 8 - Operadores de Comparação

Operadores de comparação

Os operadores de comparação utilizam símbolos para comparar expressões. Ao fazer a comparação, o programa retorna um valor booleano (True ou False). Vamos supor que você faça a seguinte comparação entre dois valores: Você se pergunta: "3 é maior que 1?" Então você analisa, e dirá verdadeiro ou falso. É isso que um programa faz, avalia a expressão e retorna verdadeiro (True) ou falso (False). Vamos ver melhor como esses operadores funcionam.

Operador Maior Que (>)

Compara se o valor da esquerda é MAIOR que o valor da direita. Vamos ver alguns exemplos:

maior_que.py 
5 > 9 # 5 é maior que 9? False
4 > 1 # 4 é maior que 1? True
10 > 0 # 10 é maior que 0? True
Saída:
False
True
True

Operador Menor Que (<)

Compara se o valor da esquerda é MENOR que o valor da direita. Veja os exemplos:

menor_que.py 
3 < 10 # 3 é menor que 10? True
5 < 2 # 5 é menor que 2? False
8 < 15 # 8 é menor que 15? True
Saída:
True
False
True

Operador Maior ou Igual Que (>=)

Compara se o valor da esquerda é MAIOR OU IGUAL que o valor da direita. Exemplos:

maior_igual.py 
5 >= 10 # 5 é maior ou igual a 10? False
10 >= 10 # 10 é maior ou igual a 10? True
15 >= 5 # 15 é maior ou igual a 5? True
Saída:
False
True
True

Operador Menor ou Igual Que (<=)

Compara se o valor da esquerda é MENOR OU IGUAL que o valor da direita. Vamos ver:

menor_igual.py 
8 <= 10 # 8 é menor ou igual a 10? True
10 <= 5 # 10 é menor ou igual a 5? False
4 <= 1 # 4 é menor ou igual a 1? False
Saída:
True
False
False

Operador Diferente de (!=)

Compara se o valor da esquerda é DIFERENTE do valor da direita. Em outras palavras, comparam se ambos são diferentes:

diferente.py 
10 != 5 # 10 é diferente de 5? True
5 != 5 # 5 é diferente de 5? False
3 != 7 # 3 é diferente de 7? True
Saída:
True
False
True

igual.py 
6 == 7 # 6 é igual a 7? False

False

Operadores lógicos

Unem as expressões condicionais. Por exemplo, você pode unir operadores de comparação. Vamos ver como isso funciona:

Operador AND

Compara se a expressão da esquerda e da direita são verdadeiras. Se ambas as expressões forem verdadeiras, então retornará True. Mas, se ao menos uma expressão for falsa, então o programa retornará False.

and.py 
5 == 5 and 12 > 8 # True and True → True
12 < 8 and 5 != 3 # False and True → False
6 > 3 and 8 > 10 # True and False → False
Saída:
True
False
False

Resumo: As duas expressões precisam ser True para retornar True, caso contrário retornará False.

Operador OR

Compara se o valor da esquerda e o valor da direita são verdadeiras. Se ambas forem falsas, então retornará False. Mas para retornar True, ao menos uma expressão deve ser True.

or.py 
7 < 10 or 6 == 6 # True or True → True
4 == 5 or 9 > 5 # False or True → True
9 != 9 or 4 < 4 # False or False → False
Saída:
True
True
False

Resumo: Ao menos uma expressão deverá ser True para retornar True. Retornará False quando ambas as expressões forem falsas.

Operador NOT

O NOT é o operador que nega um booleano, ou seja, você utilizar uma negação num True, ele será alterado para False e vice-versa.

not.py 
not True # False
not False # True
not 12 > 5 # False (era True, mas foi invertido)
not 5 < 1 # True (era False, mas foi invertido)
Saída:
False
True
False
True

Resumo: Inverte um booleano (True vira False e False vira True).
